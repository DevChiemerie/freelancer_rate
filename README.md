# Freelancer Rates
A simple JavaScript project that calculates freelancer rates based on hourly pricing, budget constraints, and monthly discounts.

This project is part of the Exercism JavaScript track and focuses on writing clean, reusable functions for real-world pricing scenarios.

## Features
* Calculate daily rate from hourly rate
* Estimate how many days a budget can cover
* Apply discounts for long-term (monthly) projects
* Proper rounding using `Math.floor` and `Math.ceil`

## Concepts Covered
* Functions and modular code
* Arithmetic operations
* Rounding numbers in JavaScript
* Problem decomposition
* Real-world pricing logic

## Tech Stack
* JavaScript (ES6)
* Node.js (for running locally)
